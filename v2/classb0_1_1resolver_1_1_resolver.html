◆ setMinimumHeartbeatInterval()

void b0::resolver::Resolver::setMinimumHeartbeatInterval ( int64_t  interval)

Set the minimum interval at which the heartbeat message must be sent by the nodes.

If the node fails to send a heartmeat message at least once in every interval, it will be considered as dead. A value of zero will disable online monitoring.

Member Data Documentation

◆ minimum_heartbeat_interval_resolver_

int64_t b0::resolver::Resolver::minimum_heartbeat_interval_resolver_
protected

The minimum interval in which the node has to send a heartbeat message. If the node fails to send a heartmeat message at least once in every interval, it will be considered as dead. A value of zero will disable online monitoring.
