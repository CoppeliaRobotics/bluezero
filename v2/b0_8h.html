void b0::init ( int &  argc,
char **  argv 
)

Initialize the B0 library.

This function will read any relevant environment variables and command line arguments, and change settings in the global data structure.

Depending if you want to override those settings or just provide a different default, you need to alter such settings before or after calling b0::init() respectively.

This function will validate command line arguments and terminate the process in case of error.
